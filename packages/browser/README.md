# browser

![npm (scoped)](https://img.shields.io/npm/v/@olmokit/browser?style=flat-square&color=FF52DE)

A set of multi-purpose low level utilities to interact with browsers APIs.

```js
import { ... } from "@olmokit/browser";
```
